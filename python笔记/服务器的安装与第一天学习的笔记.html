服务器的安装与第一天学习的笔记

查看python系统是否存在:右单击我的电脑或者计算机（根据电脑的不同而不同），选择-属性，在弹出的对话框的左边选择-高级系统设置，在高级系统设置对话框中选择环境变量选项，再在环境变量对话框的下方的系统变量中随意单击一下输入path一词，看Python是否存在
查看python系统是否存在:Ctrl+r （进入交互模式）打开命令行工具，输入cmd,回车，再输入python，回车，查看系统是否存在
查看IP：右单击电脑屏幕右下方的网络，选择-打开网络和共享中心，在网络和共享中心对话框中选择-本地连接选项，在本地连接对话框中选择-详细信息选项，在详细信息对话框中查看IPv4或者IPv6，就是本台电脑的IP
服务器的创建与数据的传输：先下载一个VisualSVN_Server_3.5.0.0_x64.1457319621安装包，把VisualSVN Server Manager从开始菜单里创建快捷方式（右单击VisualSVN Server Manager 发送到桌面快捷方式）放在桌面上，
